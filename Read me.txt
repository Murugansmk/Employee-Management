How to run this project
1. Clone the project file from github. (https://github.com/Murugansmk/Employee-Management.git)
2. Extract the file and copy employee_management folder
3.Paste inside root directory(for xampp xamp/htdocs, for wamp wampp/www,for lamp var/www/html)
4. Open phpmyadmin (http://localhost/phpmyadmin)
5. Create a database with name employee_1
6. Import employee_1.sql file(given inside the zip package in sql file folder)
7.Run the script http://localhost/employee_management/
